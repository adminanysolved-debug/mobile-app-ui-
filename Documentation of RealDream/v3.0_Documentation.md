# RealDream v3.0 Documentation - The "Admin & Control" Update

## Overview
Version 3.0 introduced a comprehensive Admin ecosystem, giving the platform owners absolute control over the app's content, users, and marketplace. This version shifted the project from a community-only app to a managed platform.

## Key Features

### 1. Unified Admin Dashboard
- **Real-time Stats**: Track total users, active vendors, dreams created, and marketplace transactions.
- **Global Overview**: Quick access to all moderation tools from a single web interface.

### 2. User & Moderation System
- **User Management**: Admins can search users, view profiles, and deactivate/activate accounts.
- **Admin User Control**: Support for multiple admin accounts with role-based access.

### 3. Marketplace & Vendor Management
- **Vendor Review**: Admins can approve or reject vendor applications.
- **Product Sync**: Ability to manage market items, ensuring only valid products are listed.
- **Purchase Tracking**: Monitor transactions across the platform.

### 4. Theme Management & Monetization
- **Theme Creator**: Admins can define new app themes (colors, styles) directly from the dashboard.
- **Pricing Control**: Set coin prices for premium themes and toggle their availability in the mobile store.

### 5. Social Content Control
- **Feed Moderation**: Ability to delete inappropriate posts or comments.
- **Gallery Management**: Manage user-uploaded images in the community gallery.

## Technical Changes
- Created `admin-system` directory with a dedicated Vite + React + Tailwind frontend.
- Implemented `admin-server` for secure backend operations.
- Added JWT-based Admin authentication.
- Shared database schema between mobile and admin backends.
